Nodes

Graph nodes are backend operations with frontend canvas controls

The node catalog is registered on the backend and requested by the frontend. Every node evaluates to an ImageFrame, and edges define which upstream frames feed each input socket. 

NodeDefinition("Merge", "Merge", "Merge", MergeNode(), ("a", "b", "mask"))
NodeDefinition("Viewer", "Viewer", "Output", ViewerNode(), tuple(str(i) for i in range(10)))

Evaluation Contract

evaluate(node: Node, inputs: dict[str, ImageFrame], context: EvaluationContext) -> ImageFrame
Inputs

Incoming edges are evaluated before the node itself.

Params

Knobs live in node.params and are part of the cache signature.

Output

Each operation returns a new ImageFrame with image data and metadata.

Core Nodes

Read

Loads files, resolves sequences, parses EXR metadata, data windows, pixel aspect, and optional channel groups.

Merge

Combines A and B inputs with over, under, plus, difference, screen, multiply, copy, matte, and other operations.

Reformat

Bilinear float resize with bbox/data-window tracking and sparse data-window optimization.

Viewer

Numbered inputs, active-input evaluation, float preview cache, GPU display path, and playback cache feedback.
